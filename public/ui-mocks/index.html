Live Caption UI Mocks

개인 모바일 사용을 우선으로 잡은 네 가지 정적 목업입니다. 기능은 없고 실제 자막 샘플이 들어간 화면 구성만 비교합니다.

Option A

Reader

읽기 편한 모바일 리더형. 번역을 기사처럼 편하게 보고 원문은 아래에 작게 둡니다.

Option B

Caption Focus

고대비 큰 자막형. 발표장/이동 중에도 한국어 번역만 빠르게 읽는 화면입니다.

Option C

Compare

번역과 원문을 함께 보는 비교형. PC 30% 사용까지 고려한 균형형입니다.

Option D

Compact

최대한 도구답게 정리한 컴팩트형. 작은 화면에서 정보 밀도를 높입니다.
